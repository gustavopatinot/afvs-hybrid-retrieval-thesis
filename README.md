# AFVS Hybrid Retrieval Thesis

Master's thesis project on adaptive filtered vector search for hybrid enterprise/RAG retrieval.

## Repository structure
- `docs/` thesis manuscript and appendices
- `references/` search strategy, PRISMA materials, study matrix
- `experiments/` experiment configurations
- `notebooks/` Colab-ready notebooks
- `src/` reusable code
- `reports/` tables, figures, and logs
- `tests/` basic validation tests
- `.github/` issue forms, PR template, and workflows

## Core traceability chain
Problem -> Issue -> Branch -> Pull Request -> Experiment -> Artifact -> Thesis Section
